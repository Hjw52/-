# 仿掘金风格的个人博客系统

采用Node全栈搭建的个人博客系统，主要采用简单风格，项目目前还在完善中~~~

### 技术栈

- 后端：egg.js （node MVC）[官方文档](https://eggjs.org/zh-cn/)
- 前端：nuxt.js （服务端渲染）[官方文档](https://zh.nuxtjs.org/guide/)
- 数据库：mongoose (mongoDB node版) [GitHub文档](https://github.com/eggjs/egg-mongoose)
### 运行

```
npm run dev
```
